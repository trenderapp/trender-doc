# Trender Documentation


Welcome to the Trender documentation, everything you need to build your own apps or learn more about the project is here.
  

## How To Contribute

Just write to the official [Trender account](https://trenderapp.com/trender) or join the [Discord server](https://discord.gg/eGpjpW3UgU)

### Bugs

If you find bugs, vulnerabilities or other think, open and issue [here](https://forms.trenderapp.com)

### Official NodeJS client

Official Client write in TypeScript and maintaint by Trender team

[https://github.com/trenderapp/trender-client](https://github.com/trenderapp/trender-client)
